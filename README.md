## Setup

- Application Launcher: `rofi`
- Bar: `polybar`
- Music Player:  `ncmpcpp`
- Notifications: `dunst`
- Shell: `zsh`
- Terminal Emulator: `urxvt`
- Text Editor: `vim`
- Video Player: `mpv`
- Web Browser: `qutebrowser`
- Window Manager: `openbox`
